# ❗ 立即操作 - 解决"未连接"问题

## 🔴 问题原因
后端有大量连接泄漏（CLOSE_WAIT），需要重启后端服务。

已杀掉旧的后端进程。

---

## ✅ 解决方案（请立即执行）

### 步骤1：打开CMD启动后端

1. 按 **Win + R**
2. 输入 **cmd**
3. 回车
4. 执行以下命令：

```bash
cd /d "D:\校招比赛\backend"
python run_backend.py
```

等待看到：
```
✅ VoicePC Backend is ready!
```

---

### 步骤2：刷新前端页面

按 **F5** 刷新 http://localhost:5174

---

## ✅ 预期结果

- ✅ 右上角显示 **"已连接"**（绿色点）
- ✅ 可以在输入框输入"你好"测试
- ✅ AI正常响应

---

## 🎯 如果还是未连接

### 方法1：使用启动脚本
双击运行项目根目录的 **`快速启动.bat`**

### 方法2：检查端口
```bash
netstat -ano | findstr "8000"
```

应该只有一个LISTENING，没有CLOSE_WAIT

---

## 📌 关键点

1. **必须用CMD**，不要用PowerShell
2. **必须等后端完全启动**再刷新前端
3. **看到"Backend is ready"**才算成功

---

**现在请打开CMD执行上面的步骤1！** 🚀
