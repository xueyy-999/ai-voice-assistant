# 🚀 如何查看VoicePC项目效果

## 方法1：手动启动（推荐）

### 第1步：启动后端
1. 打开**第一个**命令行窗口
2. 执行以下命令：
```bash
cd backend
python run_backend.py
```
3. 看到 `✅ 服务启动成功` 即可

### 第2步：启动前端
1. 打开**第二个**命令行窗口（不要关闭第一个）
2. 执行以下命令：
```bash
cd frontend
npm run dev
```
3. 看到 `Local: http://localhost:5174/` 即可

### 第3步：打开浏览器
访问：http://localhost:5174

## ❓ 常见问题

### Q: 网页打不开？
**检查步骤**：
1. 确认后端已启动（看到 ✅ 服务启动成功）
2. 确认前端已启动（看到 Local: http://localhost:5174/）
3. 确认端口未被占用
4. 刷新浏览器页面

### Q: 后端启动失败？
**解决方案**：
```bash
# 1. 检查Python版本（需要3.11+）
python --version

# 2. 安装依赖
cd backend
pip install -r requirements.txt

# 3. 重新启动
python run_backend.py
```

### Q: 前端启动失败？
**解决方案**：
```bash
# 1. 检查Node.js（需要20+）
node --version

# 2. 安装依赖
cd frontend
npm install

# 3. 重新启动
npm run dev
```

### Q: 端口被占用？
**解决方案**：
```bash
# 杀死8000端口（后端）
npx kill-port 8000

# 杀死5174端口（前端）
npx kill-port 5174
```
