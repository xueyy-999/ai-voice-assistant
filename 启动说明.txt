==============================================
VoicePC - 快速启动说明
==============================================

方式一：使用快速启动脚本（推荐）
----------------------------------------------
双击运行：快速启动.bat


方式二：手动启动
----------------------------------------------

1. 打开第一个终端（PowerShell或CMD）：
   cd D:\校招比赛\backend
   python app/main.py

2. 打开第二个终端：
   cd D:\校招比赛\frontend
   npm run dev

3. 浏览器访问：http://localhost:5173/


第一次运行前需要：
----------------------------------------------
1. 安装后端依赖：
   cd backend
   pip install -r requirements.txt

2. 安装前端依赖：
   cd frontend
   npm install

3. 配置API密钥：
   复制 backend/.env.example 为 backend/.env
   填入DeepSeek和阿里云的API密钥


详细安装说明请查看：INSTALL.md
