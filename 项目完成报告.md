## 🚀 快速启动

### 1. 安装依赖

```bash
# 后端
cd backend
pip install -r requirements.txt

# 前端
cd frontend
npm install
```

### 2. 配置API（可选）

编辑 `backend/.env`:
```env
# 如果有DeepSeek API Key，填入以启用完整AI功能
DEEPSEEK_API_KEY=your_api_key_here

# 如果有阿里云语音，填入以启用真实语音识别
ALI_APPKEY=your_appkey
ALI_ACCESS_KEY=your_access_key
ALI_SECRET_KEY=your_secret_key
```

**注意**: 即使不配置，系统也能运行（使用简化模式）

### 3. 启动服务

**终端1（后端）**:
```bash
cd backend
python app/main.py
```

**终端2（前端）**:
```bash
cd frontend
npm run dev
```

### 4. 访问应用

浏览器打开: http://localhost:5173/

## 🎬 使用演示

### 基础操作
1. **点击麦克风按钮**（或按住空格键）
2. **说出指令**: "打开记事本"
3. **松开按钮**: AI执行并语音反馈
4. **查看流程**: 点击"流程"标签查看执行步骤

### 复杂场景
1. 说 **"准备工作"**
2. 系统自动执行4个步骤
3. 实时显示执行进度
4. 完成后语音报告

### 智能对话
1. 先说 **"打开微信"**
2. 再说 **"关闭它"** ← AI能理解"它"指代微信
3. 或说 **"再打开一次"** ← AI记住上下文
