## 下一步：安装Node.js并启动前端

### 选项1：安装Node.js（推荐）

1. **下载安装**
   - 访问：https://nodejs.org/zh-cn/
   - 下载LTS版本（18.x或20.x）
   - 安装时勾选"Add to PATH"

2. **验证安装**
   ```bash
   node --version
   npm --version
   ```

3. **启动前端**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
